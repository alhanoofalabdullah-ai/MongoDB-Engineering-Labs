# Project 01 – MongoDB CRUD Operations Lab

Basic MongoDB lab covering Create, Read, Update, and Delete operations using MongoDB documents.

---

## Overview

This project demonstrates the fundamentals of MongoDB CRUD operations.

The lab covers:

- Create Database
- Insert Documents
- Find Documents
- Update Documents
- Delete Documents
- Query Filtering
- Sorting and Limiting Results

---

## Business Scenario

A company needs a simple NoSQL database to store customer information.

MongoDB is used because it stores flexible JSON-like documents and supports fast querying.
